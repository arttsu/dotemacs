* My Emacs Config
** Setup
1. Clone the repo and copy the example local variables file.
   #+begin_src shell
     git clone git@github.com:arttsu/dotemacs.git ~/.emacs.d
     cp ~/.emacs.d/local.example.el ~/.emacs.d/local.el
   #+end_src

2. Edit =local.el=.

*** Jinx
**** macOS
1. Install Enchant:
#+begin_src shell
  brew install enchant
#+end_src

2. Set =my-jinx-languages= in =local.el=:
   #+begin_src elisp
     (setq my-jinx-languages "en_US ru de_DE en_ES")
   #+end_src

*** Ledger

Install =ledger= via brew.
